# WebPlanningTools

A set of tools to plan projects, with a focus on programming projects.

Features include:
    - Basic SCRUM board like feature
        - Instead of official SCRUM boards, the boards can be created, named and archived at will
    - Basic Use Case diagram editor
    - Discussion threads linked to tasks of the SCRUM board
    - Overview of completed items over time. Note that a decrease or increase of items over time, if no other variables change, may mean an inconsisten size for your tasks, rather then improvement or decrease of efficiency.


Fun things:
    - Sticky notes
    - Drawing tool for on the sticky notes
    - As admin, you can burn archived boards to remove them. Complete with sound effect
